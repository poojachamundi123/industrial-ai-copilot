# 🏭 Industrial Knowledge AI Copilot

An AI-powered Streamlit application that allows users to upload Industrial Safety PDF documents and ask questions about their contents.

# Features

* Upload Industrial Safety PDFs
* Extract text from PDF documents
* Ask questions about:

* Safety Goals
* Employee Responsibilities
* Personal Protective Equipment (PPE)
* Emergency Action Plans
* Safety Discipline
* Simple and interactive Streamlit interface

# Technologies Used

* Python
* Streamlit
* PyPDF2

# Installation

3. Run the application:
